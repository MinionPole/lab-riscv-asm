# lab-riscv-asm

Первое лабораторное задание Lab RISC-V asm. Юнусов Роман

## Вариант задания

Вариант задания | 
--------------- | 
Найти количество отрицательных элементов в массиве. (N = 5, M = 4) |

## Результаты работы

* **example.S** - программа на RISC-V ASM
* **main.c** - программа на Си
* **main.dump** - дамп-файл программы на Си

## Сборка
Сборка примера на Си в полностью настроенном окружении выполняется с использованием консольных команд прописанных в конце файла main.c
